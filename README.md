Project idea: Implement Deep Learning to detect hand gestures to control electrnic devices

How to run:
- Step 1: cone the repo: git clone https://github.com/dunh-real/devices-controlling-by-hand-gestures
- Step 2: install dependencies: pip install -r requirements.txt
- Step 3: cd to the backend folder to run the application: cd backend
- Step 4: run the application: python app.py
